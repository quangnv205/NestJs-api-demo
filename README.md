## Description
Basic api using NestJs


## Installation

```bash
$ npm install
```
## Create command
$ nest g controller cats
$ nest g module cats 


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
